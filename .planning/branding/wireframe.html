Component Notes

Surface Hierarchy

Elevation signaled by surface token (bg-0 → bg-4), never by shadow. Nested cards step one level higher. Hover applies bg-hover.

Status Badges

Semantic lifecycle state rendered as monospace pill with 20% tinted background. Never plain prose. Consistent across tables, cards, graph nodes.

Transitions

Global 0.15s ease on bg, color, border. No transforms, no opacity toggles. Theme swap via data-theme attribute — instant.
