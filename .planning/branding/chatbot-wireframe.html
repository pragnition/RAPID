Component Notes

Chat vs Run

Chats always render a composer at the bottom. Runs never do. The presence or absence of the composer is the single load-bearing signal separating the two modes.

Inline Tool Cards

Every tool use renders in-line in the message flow. Spinner → checkmark when complete. Click to expand for args + result. No modal popovers — tools are part of the conversation.

Auto-scroll Opt-out

Messages auto-scroll to bottom by default. Scrolling up disables it; a "↓ N new messages" pill appears to jump back to live.
