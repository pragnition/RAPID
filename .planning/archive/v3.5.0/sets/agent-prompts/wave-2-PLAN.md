### Task 2: Audit all agents for phantom commands

**Action:** Run a comprehensive audit that checks every `<tools>` section across all 26 agents. For each agent, verify that every tool key referenced in its `<tools>` section exists in TOOL_REGISTRY.

**Verification script:**
```bash
cd /home/kek/Projects/RAPID && node -e "
const fs = require('fs');
const path = require('path');
const {TOOL_REGISTRY} = require('./src/lib/tool-docs.cjs');

const agentsDir = path.join(__dirname, 'agents');
const files = fs.readdirSync(agentsDir).filter(f => f.endsWith('.md'));
let errors = 0;

for (const file of files) {
  const content = fs.readFileSync(path.join(agentsDir, file), 'utf-8');
  const match = content.match(/<tools>([\s\S]*?)<\/tools>/);
  if (!match) continue;

  const lines = match[1].split('\n').filter(l => l.trim() && !l.startsWith('#'));
  for (const line of lines) {
    const keyMatch = line.match(/^\s+(\S+):/);
    if (!keyMatch) continue;
    const key = keyMatch[1];
    if (!(key in TOOL_REGISTRY)) {
      console.error('PHANTOM: ' + file + ' references nonexistent key: ' + key);
      errors++;
    }
  }
}

if (errors > 0) {
  console.error(errors + ' phantom command(s) found');
  process.exit(1);
} else {
  console.log('OK: All agent tool references exist in TOOL_REGISTRY (' + files.length + ' agents checked)');
}
"
```

**Do NOT manually edit any generated agent files.** If phantoms are found, the fix belongs in `src/lib/tool-docs.cjs` (TOOL_REGISTRY or ROLE_TOOL_MAP), and build-agents must be re-run.
