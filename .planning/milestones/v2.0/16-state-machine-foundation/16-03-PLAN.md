    - validateHandoff(agentOutput) parses RAPID:RETURN marker, then validates JSON payload with Zod schema
    - validateHandoff returns { valid: true, data } for valid COMPLETE/CHECKPOINT/BLOCKED payloads
    - validateHandoff returns { valid: false, error } for missing marker, invalid JSON, or schema violations
    - ReturnSchemas.Complete validates: status='COMPLETE', artifacts (string[]), tasks_completed (number), tasks_total (number), optional: commits, duration_minutes, next_action, warnings, notes
    - ReturnSchemas.Checkpoint validates: status='CHECKPOINT', handoff_done (string), handoff_remaining (string), handoff_resume (string), optional: artifacts, tasks_completed, tasks_total, duration_minutes, decisions, blockers
    - ReturnSchemas.Blocked validates: status='BLOCKED', blocker_category (enum), blocker (string), resolution (string), optional: artifacts, tasks_completed, tasks_total
    - ReturnSchemas.Any is a z.discriminatedUnion on 'status' field covering all three
    - Existing parseReturn, generateReturn, validateReturn still work unchanged
    - validateHandoff gives Zod-level error paths for invalid fields (e.g., "tasks_completed: Expected number, received string")
  </behavior>
  <action>
    RED: Add new describe blocks to returns.test.cjs for validateHandoff and ReturnSchemas. Do NOT modify existing tests -- append new sections. Run -- MUST fail.

    GREEN: Add to returns.cjs (do NOT modify existing functions):

    1. At top of file, add: `const { z } = require('zod');`

    2. Define Zod schemas (not exported individually, grouped under ReturnSchemas):
       - CompleteReturn = z.object({ status: z.literal('COMPLETE'), artifacts: z.array(z.string()), tasks_completed: z.number(), tasks_total: z.number(), commits: z.array(z.string()).optional(), duration_minutes: z.number().optional(), next_action: z.string().optional(), warnings: z.array(z.string()).optional(), notes: z.string().optional() })
       - CheckpointReturn = z.object({ status: z.literal('CHECKPOINT'), handoff_done: z.string(), handoff_remaining: z.string(), handoff_resume: z.string(), artifacts: z.array(z.string()).optional(), tasks_completed: z.number().optional(), tasks_total: z.number().optional(), duration_minutes: z.number().optional(), decisions: z.array(z.string()).optional(), blockers: z.array(z.string()).optional() })
       - BlockedReturn = z.object({ status: z.literal('BLOCKED'), blocker_category: z.enum(['DEPENDENCY', 'PERMISSION', 'CLARIFICATION', 'ERROR']), blocker: z.string(), resolution: z.string(), artifacts: z.array(z.string()).optional(), tasks_completed: z.number().optional(), tasks_total: z.number().optional() })
       - AnyReturn = z.discriminatedUnion('status', [CompleteReturn, CheckpointReturn, BlockedReturn])

       - Return { valid: true, data: zodResult.data }
