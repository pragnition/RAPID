    - parseHandoff(handoffContent): returns { set, pauseCycle, tasksCompleted, tasksTotal, completedWork, remainingWork, resumeInstructions, decisions }

  <action>
    **In execute.cjs, add these new functions:**

    1. `generateHandoff(checkpointData, setName, pauseCycle)`:
       - Build YAML frontmatter with: set, paused_at (ISO string), pause_cycle, tasks_completed, tasks_total
       - Build Markdown sections:
         - `## Completed Work` from checkpointData.handoff_done (default: "(none recorded)")
         - `## Remaining Work` from checkpointData.handoff_remaining (default: "(none recorded)")
         - `## Resume Instructions` from checkpointData.handoff_resume (default: "Continue from where execution stopped.")
         - `## Decisions Made` from checkpointData.decisions array (only if non-empty)
       - Return the full Markdown string

       - Return null if content is empty/falsy
       - Parse YAML frontmatter between `---` markers to extract: set, pause_cycle, tasks_completed, tasks_total
       - Parse Markdown sections: extract text under each `## ` heading
       - Return structured object: `{ set, pauseCycle, tasksCompleted, tasksTotal, completedWork, remainingWork, resumeInstructions, decisions }`

    3. `reconcileWave(cwd, waveNum, dagJson, registry)`:
       - Get wave sets from dagJson.waves[waveNum].sets
       - For each set:
         a. **Contract compliance (hard block):** Check if `.planning/sets/{set}/contract.test.cjs` exists. If so, run it via `execSync('node --test "path"', { cwd: cwd, stdio: 'pipe', timeout: 30000 })`. Run from main project root cwd (per Open Question 1 from RESEARCH.md). If test fails, add to hardBlocks with type 'contract_violation'.

         c. **Ownership compliance:** Use existing `verifySetExecution()` or `contract.checkOwnership()` to verify no cross-set file modifications. Violations are hard blocks.
         d. **Commit count:** Use `getCommitCount()` to get actual commits on the set's branch vs baseBranch.
       - Build setResults object per set: `{ contractCompliance, artifactsPlanned, artifactsDelivered, missingArtifacts, commitCount }`
       - Determine overall result: 'PASS' if no hard or soft blocks, 'PASS_WITH_WARNINGS' if only soft blocks, 'FAIL' if any hard blocks
       - Return: `{ hardBlocks, softBlocks, setResults, overall }`
