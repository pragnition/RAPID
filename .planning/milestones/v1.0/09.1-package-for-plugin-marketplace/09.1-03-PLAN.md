<!-- Marketplace JSON format from research -->

Self-hosted marketplace file structure:
- File location: `.claude-plugin/marketplace.json` at the REPOSITORY ROOT (not inside rapid/)
- This is separate from the plugin's own `.claude-plugin/plugin.json` which lives inside rapid/
- The repo root `.claude-plugin/marketplace.json` makes the entire repo a marketplace
- Users add it via: `/plugin marketplace add fishjojo1/RAPID`

marketplace.json format:
```json
{
  "name": "rapid-marketplace",
  "owner": {
    "name": "fishjojo1"
  },
  "plugins": [
    {
      "name": "rapid",
      "source": "./rapid",
      "description": "...",
      "version": "1.0.0",
      "author": { "name": "fishjojo1" },
      "category": "development",
      "tags": [...]
    }
  ]
}
```

Plugin validation command: `cd rapid && claude plugin validate .`

Official submission: Submit via in-app form at claude.ai/settings/plugins/submit or platform.claude.com/plugins/submit
