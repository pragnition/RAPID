         - Team configuration section (team size, parallel capacity)
         - Key Decisions table (empty, ready to be populated)

       - `detectExisting(cwd)`: Check if `.planning/` directory exists at cwd. If exists, list files inside it. Return `{ exists: boolean, files: string[] }`.

         - `'upgrade'`: Check each expected file; only write if missing. Return `{ created: [...newFiles], skipped: [...existingFiles] }`.
         - `'cancel'`: Return `{ cancelled: true }`.
         - **CRITICAL:** Do NOT create `.planning/phases/` -- those are created on-demand during planning.
