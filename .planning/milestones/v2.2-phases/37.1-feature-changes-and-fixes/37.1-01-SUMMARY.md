## Decisions Made
- quickTasks uses `z.array(QuickTask).optional().default([])` -- existing STATE.json files without quickTasks parse cleanly, defaulting to empty array
- Quick task ID auto-increment uses max(existing IDs) + 1 rather than array length -- handles gaps from future deletion
- Migration detection cascading priority: GSD (gsd_state_version, phases/ with PLAN.md) > openspec (SPEC.md with openspec content) > generic (STATE.md/ROADMAP.md) > none
- backupPlanning throws Error if .planning.bak/ already exists -- prevents silent data loss
- Quick task functions are async even though currently synchronous -- prepared for proper-lockfile integration in future
