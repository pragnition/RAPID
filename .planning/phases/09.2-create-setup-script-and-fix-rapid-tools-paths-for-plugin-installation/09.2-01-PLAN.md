  <name>Task 1: Create setup.sh bootstrap script</name>
  <files>setup.sh</files>
  <action>
Create setup.sh at the repo root. This is a bash script (#!/usr/bin/env bash) with set -euo pipefail.

**Self-location:** Use `SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"` (NOT readlink -f, which fails on macOS). Derive `RAPID_TOOLS_PATH="$SCRIPT_DIR/src/bin/rapid-tools.cjs"`.

**5 idempotent steps:**

Step 1 - Check prerequisites:
- Verify `node` exists and is v18+. Parse version with `node -v | sed 's/v//' | cut -d. -f1`.
- Verify `git` exists. Print versions on success.
- Exit 1 with clear error if either missing or too old.

Step 2 - Install npm dependencies:
- Skip if `$SCRIPT_DIR/node_modules` already exists.
- Otherwise run `(cd "$SCRIPT_DIR" && npm install --production)`.

Step 3 - Validate rapid-tools.cjs:
- Run `node "$RAPID_TOOLS_PATH" prereqs --json` and check exit code.
- Exit 1 if validation fails.

Step 4 - Persist RAPID_TOOLS env var:
- Export RAPID_TOOLS in current session: `export RAPID_TOOLS="$RAPID_TOOLS_PATH"`
- Detect shell via `$SHELL` env var basename:
  - fish: config file = `$HOME/.config/fish/config.fish`, syntax = `set -gx RAPID_TOOLS "path"`
  - zsh: config file = `$HOME/.zshrc`
  - bash: config file = `$HOME/.bashrc` (fallback to `.bash_profile` if .bashrc doesn't exist)
  - other: config file = `$HOME/.profile`
  - For non-fish: syntax = `export RAPID_TOOLS="path"`
- Idempotent: grep for "RAPID_TOOLS" in config file first. Skip if found.
- If not found, append with a comment header "# RAPID plugin for Claude Code".
- For fish, ensure `$HOME/.config/fish/` directory exists (create with mkdir -p if not).

Step 5 - Register Claude Code plugin:
- Check if `claude` CLI exists. If not, print skip message with manual instructions.
- If exists, run `(cd "$SCRIPT_DIR" && claude plugin add . 2>&1)`. Treat failure as non-fatal warning.

**Output:** Print summary with RAPID_TOOLS path and source command for shell config.

**Installation method detection:** Add a function `detect_install_method()` that checks if `$SCRIPT_DIR` contains `.claude/plugins` -- returns "marketplace" or "clone". Print this in the summary header.

Make the file executable (chmod +x).
